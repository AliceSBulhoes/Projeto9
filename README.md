# Comandos Git

### Criando uma Branch

<p>git checkout -b "NOME DA BRANCH"</p>

### Verificando Branch existente

<p>git branch</p>

### Fazemdo Merge das Branchs

*Voltar para branch master antes de digitar*

<p>git merge "NOME DA BRANCH"</p>
